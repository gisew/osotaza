Oso Animado Interactivo

¿Cómo lo hice?

Este oso animado fue creado usando **Blender 3D**, donde modelé y texturicé el oso, lo animé para que interactúe con la paleta y lo exporté como un archivo `.glb`.

Luego, utilicé **HTML** y el componente "model viewer" para integrar el modelo 3D en la web, permitiendo una experiencia interactiva.

¡Todo está diseñado para que sea fácil de ver y jugar con el oso, mientras disfrutas de un diseño moderno y kawaii!
